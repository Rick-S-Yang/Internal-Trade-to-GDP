# Internal Trade to GDP Analysis

## Overview
This project analyzes the relationship between **internal trade and GDP** using economic data and statistical analysis in **R**.  
The objective is to understand how internal trade contributes to overall economic output and to identify key trends and patterns over time.

## Project Objectives
- Examine the relationship between internal trade activity and GDP
- Analyze trends in internal trade contribution to economic growth
- Visualize economic data to support interpretation and insights
- Apply data cleaning and analytical techniques using R

## Data & Methodology
- Collected and cleaned economic datasets related to internal trade and GDP
- Conducted exploratory data analysis and descriptive statistics
- Visualized trends and relationships using R (time-series plots and comparative charts)
- Interpreted results using economic and analytical reasoning

## Key Insights
- Internal trade shows a consistent relationship with GDP over time, highlighting its role in economic structure
- Changes in trade activity are associated with observable movements in GDP levels
- Visualization helps reveal long-term trends and cyclical patterns in trade-driven economic output
